# Практика 24ВВВ3, бригада 9
Репозиторий для материалов учебной практики бригады 9 группы 24ВВВ3.
В проекте хранится исходный код программы, инструкция по запуску, тестовые данные и материалы для проверки.

## Тема работы
Реализация алгоритма поразрядной сортировки Radix Sort.
Программа написана на языке C. Она позволяет ввести массив вручную или считать данные из файла `input.txt`, выполнить сортировку и сохранить результат в файл `output.txt`.

## Основная ветка
Основная ветка проекта: `main`.
В `main` напрямую ничего не загружаем. Все изменения сначала делаются в отдельной ветке участника, потом отправляются через Pull Request.

## Как запустить программу
1. Открыть терминал в папке проекта.
2. Перейти в папку `src`.
3. Скомпилировать программу:

```bash
gcc main.c -o radix_sort
```

4. Запустить программу:

```bash
./radix_sort
```

Если запуск выполняется в Windows, можно использовать:

```bash
gcc main.c -o radix_sort.exe
radix_sort.exe
```

## Проверка через файл
1. В папке `src` должен лежать файл `input.txt`.
2. В файле должны быть числа через пробел.
3. После запуска программы выбрать пункт `2`.
4. Результат появится в файле `output.txt`.

## Что должно быть проверено
- ручной ввод данных;
- чтение данных из файла;
- запись результата в файл;
- сортировка уже отсортированного массива;
- сортировка массива в обратном порядке;
- сортировка случайного набора чисел.
